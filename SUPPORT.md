# Support Guidelines

需要帮助或想反馈问题？请根据不同情况选择合适的渠道。

## FAQ 与文档
- 阅读 `README.md` 与 `docs/README.md` 获取安装、配置与运行信息。
- 如果文档缺失或不准确，欢迎提交 Issue 或 PR 补充。

## Bug / 技术问题
1. 搜索现有 Issues，避免重复反馈。
2. 如需创建新的 Bug Issue，请使用 “Bug report” 模板并附上日志、配置与复现步骤。
3. 安全漏洞请勿在公开 Issue 中披露，改用 `SECURITY.md` 描述的流程。

## 功能建议
- 使用 “Feature request” 模板描述业务场景、预期行为与方案讨论。
- 复杂提案可先在 GitHub Discussions 或社区渠道讨论。

## 实时沟通
- 若有紧急生产问题，请通过 Issue 并标记 `priority/high`，维护者会尽快响应。

## 贡献与评审
- 想参与项目开发，请阅读 `CONTRIBUTING.md`。
- PR 附带测试、文档与变更说明有助于加快审查进度。

感谢你对 Soffio 的关注与支持！
